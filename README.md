# 📚 Sistem Peminjaman Buku - Laravel

Website perpustakaan berbasis Laravel yang memungkinkan pengguna untuk melihat katalog buku, melakukan peminjaman, dan mengelola data buku dengan sistem role (admin & user).

---

## ✨ Fitur Utama

- 🔐 Login & Register User
- 👨‍💼 Role Admin & User
- 📚 CRUD Data Buku
- 🏷️ Kategori / Catalog Buku
- 🔍 Filter Buku berdasarkan kategori
- 📦 Peminjaman Buku
- 💰 Manajemen Denda (opsional)

---

## 🛠️ Teknologi yang Digunakan

- PHP (Laravel)
- MySQL
- Bootstrap 5
- Blade Template

---

## 📂 Struktur Fitur

- Admin Dashboard
- Manajemen Buku
- Manajemen Kategori (Catalog)
- Daftar Buku (User)
- Sistem Peminjaman
